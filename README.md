# gitpull - incorporate changes from a remote repository into the current branch

## Usage

    var gitpull = require('gitpull')
    
    gitpull('directory', function (err) {
      console.log(err || 'OK')
    })

## Installation

install via [npm](http://npmjs.org/):

    npm install gitpull
